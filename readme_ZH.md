# Homebrew

官网下载：https://brew.sh，安装命令：

```bash
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

# iTerm2

## 安装

第一种方式，官网下载 https://iterm2.com/。

第二种方式，Homebrew

```bash
$ brew install --cask iterm2
```
